Enumeration Scheduler 

public enum Scheduler

Properties

`default` 

var `default`

Scheduler.main by default. Change the scheduler to change the default behavior where callbacks attached via on method are always called on the main thread.

main 

let main: ScheduleWork

If the task finishes on the main thread, the callbacks are executed immediately. Otherwise, they are dispatched to be executed asynchronously on the main thread.

immediate 

let immediate: ScheduleWork

Immediately executes the given closure.
