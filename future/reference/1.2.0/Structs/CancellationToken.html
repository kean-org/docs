CancellationToken

public struct CancellationToken

Enables cooperative cancellation of operations.

You create a cancellation token by instantiating a CancellationTokenSource object and calling its token property. You then pass the token to any number of threads, tasks, or operations that should receive notice of cancellation. When the owning object calls cancel(), the isCancelling property on every copy of the cancellation token is set to true. The registered objects can respond in whatever manner is appropriate.

All CancellationToken methods are thread safe.

    Returns true if cancellation has been requested for this token. Returns false if the source was deallocated.
