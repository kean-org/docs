init(_:​) 

@discardableResult public init(_ closure: () -> Void)

All of the constraints created in the given closure are automatically activated at the same time. This is more efficient then installing them one-be-one. More importantly, it allows to make changes to the constraints before they are installed (e.g. change priority).
