Structure Image​Processing​Options.​Border 

public struct Border: Hashable, CustomStringConvertible

Draws a border.

To make sure that the border looks the way you expect, make sure that the images you display exactly match the size of the views in which they get displayed. If you can't guarantee that, pleasee consider adding border to a view layer. This should be your primary option regardless.
