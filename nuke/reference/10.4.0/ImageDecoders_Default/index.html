Class Image​Decoders.​Default 

public final class Default: ImageDecoding, ImageDecoderRegistering

A decoder that supports all of the formats natively supported by the system.

The decoder automatically sets the scale of the decoded images to match the scale of the screen.

The default decoder supports progressive JPEG. It produces a new preview every time it encounters a new full frame.
