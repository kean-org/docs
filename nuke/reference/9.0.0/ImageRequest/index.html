init(url:​processors:​priority:​options:​) 

public init(url: URL, processors: [ImageProcessing] = [], priority: ImageRequest.Priority = .normal, options: ImageRequestOptions = .init())

Initializes a request with the given URL.

ImageRequest allows you to set image processors, change the request priority and more:

let request = ImageRequest(
    url: URL(string: "http://...")!,
    processors: [ImageProcessors.Resize(size: imageView.bounds.size)],
    priority: .high
)

Parameters

Name Type Description
priority Image​Request.​Priority
  • priority: The priority of the request, .normal by default.

  • processors: Image processors to be applied after the image is loaded.
