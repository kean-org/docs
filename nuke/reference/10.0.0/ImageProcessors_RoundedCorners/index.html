Structure Image​Processors.​Rounded​Corners 

public struct RoundedCorners: ImageProcessing, Hashable, CustomStringConvertible

Rounds the corners of an image to the specified radius.

In order for the corners to be displayed correctly, the image must exactly match the size of the image view in which it will be displayed. See ImageProcessor.Resize for more info.
